The ImageMagick stable and development source releases are available via anonymous Subversion. This is a convenient way for developers to download the ImageMagick source, fix bugs, or add new features.

To retrieve, or check out ImageMagick stable, use this command:

  svn co https://www.imagemagick.org/subversion/ImageMagick/branches/ImageMagick-6.3.6 ImageMagick-6.3.6

To retrieve, or check out ImageMagick development, use this command:

  svn co https://www.imagemagick.org/subversion/ImageMagick/trunk ImageMagick

To update the sources within a previously checked out directory, execute:

  svn update

and only the files which have changed are updated.
